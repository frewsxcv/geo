Struct UnimplementedGeometryCollection

Source 
pub struct UnimplementedGeometryCollection<T>(/* private fields */);
Expand description

An empty struct that implements GeometryCollectionTrait.

This can be used as the GeometryCollectionType of the GeometryTrait by implementations that don’t have a GeometryCollection concept
